The Details.

The Groove is a web experience built to bring a social atmosphere to a usually isolated streaming platform, Spotify. Using NodeJS, ExpressJS, and Socket.IO this application allows users with Spotify Premium accounts to create and join Live Play Lists as well as add and delete songs to these lists. The Spotify API is used to authenticate users, display currently playing song, and search the Spotify database for songs. This web experience is hosted on an Amazon Web Server.
